## Описание программы

Book Tracker — это GUI-приложение для управления списком прочитанных книг. Программа позволяет:

- Добавлять книги (название, автор, жанр, количество страниц)
- Просматривать список книг в таблице
- Фильтровать книги по жанру и количеству страниц
- Удалять книги через контекстное меню
- Сохранять и загружать данные в формате JSON

---

## Требования

- Python 3.7 или выше
- Библиотека tkinter (входит в стандартную поставку Python)
